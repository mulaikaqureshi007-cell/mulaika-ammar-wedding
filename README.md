# Mulaika & Ammar — Wedding Website (GitHub Pages)

This is a static wedding website you can host for free with GitHub Pages.

## Edit the content
- Open `index.html` and update:
  - RSVP link: search for `REPLACE_WITH_YOUR_FORM`
  - Contact cards
  - Any “TBC / TBD” timings & venue details
- Put your photos in `/assets` and rename them:
  - `photo-1.jpg`, `photo-2.jpg`, `photo-3.jpg`, `photo-4.jpg`
  - Or update the filenames in the Gallery section in `index.html`.

## Publish on GitHub Pages
1. Create a new GitHub repo (public) and upload these files.
2. Go to **Settings → Pages**
3. Under “Build and deployment”:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
4. Your site will be live at:
   - `https://YOURUSERNAME.github.io/REPO-NAME/`

## Optional polish
- Change theme colors in `styles.css` (search for `--accent`)
- If you want to remove any section, delete it in `index.html`
